🚀 Build Application (automated)

One button generates the project, rewrites the flow's /mgmt/data/esb directories to C:\Temp\test\<project> (keeping the suffix), builds the BAR with ibmint, starts the runtime and deploys with mqsi — you supply the Queue Manager / Node / Server.

🧪 Test Application (automated)

Enter the application name; the tool drops the sample input file into the flow's input folder, and the deployed flow reads it and produces the output. The test polls the output folder — including the mqsitransit staging folder — for any *.json/*.txt written after the drop.
